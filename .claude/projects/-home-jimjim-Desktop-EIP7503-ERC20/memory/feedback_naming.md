Don't shorten variable names unless they are 4+ words long. E.g. use `syncFields` not `sf`, use `chainId` not `cid`.

**Why:** User needs to read the code and short abbreviations hurt readability.

**How to apply:** Always use full descriptive names for variables. Only abbreviate when the name would be excessively long (4+ words).
